# 介绍
在本章中提到的竞赛大多为**项目赛、创新创业比赛**。

> 大学生创新训练项目是一个特例，但也归类在《竞赛》章节中讲解。

大家能够参与到的竞赛每年都不同，所以这里只给出一些例子：

- 中国国际大学生创新大赛（互联网+）
- 挑战杯
- 创新、创业、创意大赛（三创赛）
- 计算机设计大赛（计设）
- 海峡两岸（泛珠）

这些比赛大多需要通过团队合作（或个人）完成项目，随后上交相关文档、资料供评委了解项目，最后通过汇报和答辩完成对项目介绍。

# 参与竞赛的好处

- 考研复试
- 综测加分
- 丰富简历
- 充实生活

> 别看只是寥寥几个字，实际上竞赛获奖对于大家的帮助远比想象中的要大。
